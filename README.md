# Music_Store_Analysis

##  Project Overview

This project analyzes a digital music store database using SQL to solve real-world business problems and generate actionable insights. The analysis focuses on customer behavior, sales performance, music genres, artists, and revenue trends.

Through a series of SQL queries, this project demonstrates essential data analysis techniques such as filtering, sorting, joins, aggregations, and subqueries, making it a strong portfolio project for aspiring Data Analysts.

---

## Project Objectives

- Analyze customer purchasing behavior.
- Identify top-performing artists and music genres.
- Discover sales trends across different countries and cities.
- Answer business questions using SQL.
- Practice writing efficient and well-structured SQL queries.

##  Business Questions Solved

### Easy
- Identify the senior-most employee based on job title.
- Find the countries with the highest number of invoices.
- Retrieve the top three highest invoice totals.
- Determine the city generating the highest revenue.
- Identify the customer who has spent the most money.

### Moderate
- Find all customers who purchased Rock music.
- Identify the top 10 Rock artists based on the number of tracks.
- Find tracks with a duration longer than the average song length.

### Advanced
- Calculate the amount spent by each customer on each artist.
- Identify the most popular music genre for each country.
- Determine the highest-spending customer in each country.
